# Personal Portfolio Website

A modern and responsive personal portfolio website developed using **HTML**, **CSS**, and **JavaScript**. This portfolio showcases my education, technical skills, projects, certifications, resume, and contact information.

## Features

* Responsive Design
* About Me Section
* Technical Skills
* Projects Showcase
* Resume Download
* Contact Information
* Smooth Navigation
* Modern UI Design

## Technologies Used

* HTML5
* CSS3
* JavaScript

1. Download or clone this repository.
